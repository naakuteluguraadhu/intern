The primary objectives of this project are:
1. To develop a user-friendly application that captures images and videos from a webcam.
2. To implement features such as filtering (grayscale and sepia) and timestamping for captured media.
3. To provide an interactive interface for users to control the capture process.

Tools and Technologies

  1. Programming Language: Python
  2. Libraries:
    1. OpenCV (cv2) for image and video processing
    2. NumPy for numerical operations
    3. Datetime for handling timestamps
  3. Operating System: The application is compatible with Windows, macOS, and Linux, provided Python and OpenCV are installed.

Core Components

  1. Webcam Capture:The application captures video from the default webcam using OpenCV's VideoCapture class. The capture continues until the user decides to quit or the specified number of images is reached.
  2. Image Filtering:The application allows users to apply two types of filters to the captured frames:
  3. Grayscale:Converts the image to grayscale, enhancing contrast and simplifying image data.
  4. Sepia: Applies a sepia tone, creating a warm, vintage effect. This is done using a transformation matrix to modify the pixel colors.
  5. Timestamp Addition:A timestamp showing the current date and time is added to each captured frame. This helps in identifying when the image or video was recorded.

User Interactions:

  • The application listens for specific key presses:
  • 'q': Quit the application.
  • 's': Save the current frame as an image in the specified output folder.
  • 'v': Start or stop video recording. If recording, the application saves the video in AVI format.
  • 'f': Cycle through the available filters (grayscale, sepia, none).

Usage Instructions

  1. Running the Application:
    • The user is prompted to enter the directory where images and videos will be saved.
    • The webcam feed is displayed in a window.
  2. Capturing Images:
    • Press 's' to save the current frame as an image. The images are saved in the specified output folder.
  3. Recording Videos:
    • Press 'v' to start or stop video recording. The recorded videos are saved in the specified output folder with a timestamped filename.
  4. Applying Filters:
    • Press 'f' to apply filters to the webcam feed. The filters can be cycled through (none, grayscale, sepia).
  5. Exiting the Application:
    • Press 'q' to quit the application.

RESULT

The application successfully captures images and videos from the webcam while allowing users to apply filters and add timestamps. Users can save their captured media in the desired format, making it a versatile tool for personal or professional use.
